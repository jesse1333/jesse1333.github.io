# Portfolio Website

A React + Vite portfolio website boilerplate with React Router.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Project Structure

```
src/
├── components/     # Reusable components (Header, Layout)
├── pages/          # Page components (Home, About, Projects, Contact)
├── App.jsx         # Routes configuration
├── App.css         # App-specific styles
├── main.jsx        # Entry point
└── index.css       # Global styles
```

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start dev server |
| `npm run build` | Build for production |
| `npm run preview` | Preview production build |

## Customization

- **Home** (`src/pages/Home.jsx`) – Hero / landing section
- **About** (`src/pages/About.jsx`) – Bio, skills, experience
- **Projects** (`src/pages/Projects.jsx`) – Project showcase
- **Contact** (`src/pages/Contact.jsx`) – Contact form or links

Edit `src/components/Header.jsx` to change navigation links.
